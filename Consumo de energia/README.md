#Calculadora de consumo de Energia

#Objetivo do projeto
O projeto serve como auxílio para o calculo do consumo de energia mensal de aparelhos utilizados durante o dia a dia

#Linguagem de programação utilizada
A linguagem de programação utilizada para o cálculo foi o python
