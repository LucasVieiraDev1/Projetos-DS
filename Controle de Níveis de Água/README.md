## Controle de Níveis de Água

## Objetivo do projeto:
O objetivo deste projeto é simular a verificação do nível de abastecimento de um reservatório de água, exibindo no terminal uma mensagem de alerta com cor correspondente ao nível informado.

## ⚙️Linguagem de programação utilizada
A linguagem de programação utilizada para o cálculo foi o python
