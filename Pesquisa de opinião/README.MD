## 📌Pesquisa de opinião para a empresa de marketing TudoWeb

## 🚀Objetivo do Sistema

O projeto auxília a empresa TudoWeb a ter um feedback do seu atendimento por meio do registro das notas de 1 a 3 fornecidas pelos clientes. Neste sistema foram utilizadas estruturas de repetição e decisão.

## ⚙️Linguagem de programação utilizada
A linguagem de programação utilizada para o cálculo foi o python
![Python](https://img.shields.io/badge/Python-3776AB?style=for-the-badge&logo=python&logoColor=white)

## 📊O que o projeto realiza?

1. O programa solicita os dados do usuário (nome, idade e opinião)
2. Repete o processo para vários entrevistados
3. Armazena a quantidade de respostas
4. Exibe o resultado final
